Vous connaissez Docker, non ? Mais connaissez-vous Docker Compose ? Oui ? Ok
mais savez-vous ce qu'il y a de nouveau depuis la version 2.0 ? Et savez-vous
que vous pouvez utiliser Docker Compose pour vos environnements mais également
pour la production ? Non ? Alors vous êtes au bon endroit!

Durant cette présentation, nous vous présenterons un rapide historique de Docker
Compose et vous découvrirez les nouvelles fonctionnalités Compose v2.
Nous vous montrerons des cas d'usage avancés que vous pourrez utiliser
localement pour vos développements. Mais savez vous que les Ops peuvent aussi
tirer avantage de Compose pour déployer leurs applications dans le Cloud? Oui
vous avez bien compris une seule commande Compose pour déployer toute votre
application chez votre fournisseur Cloud et sans avoir besoin de Kubernetes !
En parlant de Kubernetes, nous finirons notre session en vous donnant quelques
conseils pour choisir quand utiliser Docker Compose plutôt que K8s.
