GitOps est une nouvelle méthodologie qui a pour but d'automatiser son infrastructure en utilisant Git comme source unique de configuration.

Dans cette session de live coding, nous explorerons comment GitOps fonctionne dans la vraie vie: comment le mettre en place sur un projet existant, les bénéfices que l'on peut en attendre, et les problèmes qui peuvent survenir.

Pour illustrer cet exemple, nous utiliserons GitHub Actions, Terraform et Azure, pour déployer automatiquement un projet JHipster (Java / Spring Boot).
