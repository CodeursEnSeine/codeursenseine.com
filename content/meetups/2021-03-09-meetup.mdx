### Meetup : JHipster ❤️ Quarkus: Supersonic, Subatomic Full Stack Application

D’un côté JHipster, LE générateur d’application le plus connus de l’écosystème Java pour générer des applications fullstack.
De l’autre Quarkus, le dernier framework de Red Hat qui annonce des performances époustouflantes.
L’occasion était trop belle, rajouter un backend Quarkus à JHipster en utilisant le concept de “blueprint”.

Durant cette présentation, nous aborderons les points suivants :

- C'est quoi JHipster et surtout c'est quoi un blueprint
- C'est quoi Quarkus et pourquoi autant de hype.
- JHipster-Quarkus
- Comment l'installer
- Generation d'une app de commerce en ligne avec un backend Quarkus qui pèse 80M de mémoire et démarre en 0.0.1
- Generation d'une entité en passant par le formulaire en CLI
- L’utilisation du JDL (JHipster Description Language) pour aller encore plus vite
- Comment aller plus loin
- Le futur

Le tout sans slide, les explications se font en bougeant les mains ou en montrant du code :-)

Par les commiteurs de JHipster Quarkus
