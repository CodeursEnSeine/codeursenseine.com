### Découverte de Microsoft Fabric

Le Pôle data Devolis nous propose une session de découverte de Microsoft Fabric, la nouvelle plateforme SaaS de Microsoft pour le développement de projets BI.
Présentation globale de la solution et une démonstration pratique au travers d’un cas d’usage concret.
Rayen MEJRI & Clément Gommé

### SQL Server et notifications

Quand la base de données contribue au temps réel
En règle générale, c'est le code qui notifie la base de données;
Durant cette présentation, nous tenterons de démontrer que l'inverse est aussi possible
