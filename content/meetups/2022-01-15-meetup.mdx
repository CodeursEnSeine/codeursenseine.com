### Première application mobile Flutter ? Ne faites pas les mêmes erreurs que nous !

En version stable depuis 3 ans, le framework mobile Flutter a su s'imposer comme un incontournable dans le monde du développement mobile cross-platform. Vous en êtes sans doute conscient et vous avez peut-être décidé de coder votre première application... mais vous ne savez pas par où commencer.

Entre le choix du state management, le theming de votre application, la structure des dossiers, les tests, les librairies à utiliser et le déploiement sur les stores, vous êtes perdu Visage songeur

Avec 1 an et demi de recul sur le développement Flutter sur deux applications maintenant en production, on essaiera de vous expliquer au mieux les bases de Flutter afin de vous orienter vers les bonnes pratiques tout en vous évitant de commettre les mêmes erreurs que nous. Bref, on vous transmettra les choses que l'on aurait aimé que l'on nous dise quand on a découvert le framework.

Ce talk sera un live coding où vous verrez la construction d'une application Flutter à travers de vrais bouts de code Clavier
