### Meetup : Introduction à la nouvelle machine virtuelle GraalVM

(La présentation sera en français)

5 years later, Java is still alive and the whole java ecosystem should reinvent itself especially in the micro -service era. In this context, a bunch of innovations are taking place in java landscape, this talk is dedicated to one center-piece of a Java Program: The Virtual Machine, We will explore GraalVM and talk about the benefits it comes with: JIT, AOT, PGO, etc..
