## Meetup en association avec [Normandie.ai](https://normandie.ai)

### REX GenAI: Comment garantir un Code Propre avec GitHub Copilot (Kévin Llopis)

Développer plus vite sans sacrifier la qualité, c’est possible avec GitHub Copilot.

Dans cette conférence, nous verrons comment utiliser GitHub Copilot pour créer de nouvelles fonctionnalités de bout en bout, tout en respectant les standards et bonnes pratiques d'une équipe de développement.
À partir d'exemples concrets, nous apprendrons à structurer nos prompts pour maximiser la qualité du code généré.

### Coder avec l'IA générative : adopter un nouveau mindset ? (Aurian Guerard)

Dans ce talk, Aurian Guerard des Lauriers (Senior Full-Stack @ Pennylane) vous partage notre démarche concrète pour passer d’une simple utilisation de l’IA à un véritable changement de paradigme.
Découvrez les outils que nous avons introduits, les cas d’usage qui ont fait la différence et les bonnes pratiques que nous avons mises en place pour maximiser l’impact sans sacrifier la qualité.
