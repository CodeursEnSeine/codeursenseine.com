Le paysage du développement mobile est depuis longtemps marqué par la guerre des frameworks : applications natives, React Native, Flutter et applications web progressives (PWA).

Si les comparaisons traditionnelles s'appuient souvent sur des benchmarks synthétiques ou des exemples simplifiés à l'extrême, cette présentation propose une approche inédite : exploiter plusieurs agents de programmation IA pour créer des applications identiques, issues du monde réel, sur différents frameworks mobiles, puis évaluer les résultats à l'aide de métriques objectives.

Cette session démontrera comment divers agents IA répondent aux mêmes exigences d'une application mobile en utilisant Kotlin/Swift (natif), React Native, Flutter et des frameworks web modernes.

Nous analyserons en détail les défis rencontrés par chaque agent, la qualité du code produit et proposerons un cadre d'évaluation multidimensionnel qui dépasse les simples comparaisons superficielles.
