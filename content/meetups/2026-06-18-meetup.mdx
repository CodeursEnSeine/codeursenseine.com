📢 Pour ce meetup, Codeurs en Seine vous accueille chez **Zone01 Rouen** pour une plongée dans l'**archéologie logicielle**.

## Le sujet

En 2020, la communauté Mush a entrepris de reconstruire **from scratch** un jeu web multijoueur… sans code source ni spécifications. Trois ans d'archéologie logicielle plus tard, Charles-Meldhine revient sur cette aventure : comment extraire les règles métier à partir de sources hétérogènes — et parfois contradictoires —, et comment les **tests automatisés** sont devenus le lieu où cette connaissance s'est cristallisée.

Un retour d'expérience précieux pour quiconque doit un jour reprendre un projet legacy dont le fonctionnement est mal documenté… ou oublié.
