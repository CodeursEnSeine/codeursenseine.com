### Évaluer la consommation d'énergie d'une machine avec scaphandre

La consommation d'énergie du numérique est loin d'être la seule composante de son empreinte environnementale.
Être capable d'évaluer la consommation d'énergie de ses services ou infrastructures peut cependant s'avérer très utile pour prioriser les actions visant à réduire leur empreinte.
Nous verrons ensemble comment évaluer la consommation d'énergie d'une machine sous GNU/Linux avec Scaphandre.
Nous parlerons de ce qui se cache sous le capot et des limites de la mesure logicielle de la consommation d'énergie,
dont il faut avoir conscience, des pièges à éviter.

### Évaluer la qualité écologique de votre code avec ecoCode

Tout comme le Clean Code vise à appliquer les meilleures pratiques pour maintenir un code propre dans le temps,
le Green Code vise à appliquer les meilleures pratiques pour écrire un code + écologique et réduire sa dette.
Et il existe un outil de revue de code pour cela : ecoCode.
Nous verrons ensemble sur des exemples concrets comment celui-ci fonctionne.
