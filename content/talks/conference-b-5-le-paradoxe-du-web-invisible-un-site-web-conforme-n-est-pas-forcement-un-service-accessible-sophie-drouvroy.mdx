Nous parlons souvent de l’accessibilité numérique des sites web par leur interface utilisateur mais rarement de l’expérience utilisateur. Un site web n’est qu’une étape d’une expérience utilisateur. Est-ce qu’on peut parler d’assurance qualité accessible ? “Ce site est à 100% conforme, notre service est totalement inaccessible.”

En explorant ces défis, nous chercherons à sensibiliser les concepteurs, les développeurs et les décideurs à l'importance de dépasser les apparences et de s'efforcer d'atteindre une véritable accessibilité de services pour tous.
