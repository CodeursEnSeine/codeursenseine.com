Depuis Java 17, les vendeurs ont décidé de proposer un support commercial à long terme (LTS) tous les deux ans. Cela donne une importance particulière Jva 21 et 25, puisque ce sont celles que les projets choisissent d'utiliser. Comme à chaque fois les ajouts, correctifs et améliorations sont nombreux entre ces deux versions. On se concentrera sur ce qui nous intéresse en tant que développeur : les modifications du langage et les nouvelles API, de même que les gains en performance. On couvrira essentiellement trois points : Amber et ses quelques ajouts, L'API Stream et ses Gatherers, ainsi que Loom et le nouveau modèle de concurrence construit sur les threads virtuels. On parlera de l'avenir avec les Stable Values et les Value Types.
