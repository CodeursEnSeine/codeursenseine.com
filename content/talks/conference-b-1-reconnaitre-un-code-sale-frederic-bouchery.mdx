On parle souvent de "code propre", mais il faut avouer qu'il existe tellement de littérature sur le code propre que l'on ne sait pas toujours comment bien commencer. Certes, il y a quelques accords sur certaines pratiques, mais on peut aussi lire des approches parfois antinomiques.

"Moi, j'applique rigoureusement DRY !"
"Ha non, DRY n'est pas une bonne pratique, il est préférable d'être WET !" (non, ce n'est pas une vanne).
Du coup, nous vous proposons une autre approche pour commencer : Regardons ce qu'est un code sale.
En effet, sur certaines pratiques, il y a un certain consensus, et apprendre à éviter certaines pratiques, c'est déjà une avancée vers un code plus propre, ou tout du moins, moins sale.
