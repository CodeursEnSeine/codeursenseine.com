L’expérience utilisateur est devenue le cœur du développement web moderne. Après avoir bien navigué dans l’écosystème headless et composable, je vous invite à découvrir comment les architectures modernes peuvent changer la façon dont vous créez des sites performants, intuitifs et évolutifs.

Au programme, nous explorerons ensemble les buzzwords qui redéfinissent le web aujourd’hui : Jamstack, Composable, Headless, MACH, ...

L'objectif de cette présentation est de vous présenter :

Pourquoi l'UX est incontournable et comment ces architectures vous permettent de créer des expériences utilisateur engageantes et durables.
Comment optimiser la performance et la scalabilité avec des architectures composables, pour des sites rapides qui s'adaptent à vos besoins.
Comment adopter l'approche headless pour offrir du contenu dynamique et personnalisé, tout en simplifiant la gestion.
Ce talk est conçu pour vous donner les clés d’une création web moderne, efficace et centrée sur l’utilisateur. Mon objectif est que vous repartiez avec des idées concrètes et applicables pour intégrer ces technologies dans vos projets, afin d'améliorer l’expérience utilisateur de vos sites.
