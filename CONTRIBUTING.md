# Contributing to Codeurs en Seine

👍🎉 First off, thanks for taking the time to contribute! 🎉👍

The following is a set of guidelines for contributing to Codeurs en Seine website, which are hosted in the [CodeursEnSeine Organization](https://github.com/CodeursEnSeine) on GitHub.
These are mostly guidelines, not rules. Use your best judgment, and feel free to propose changes to this document in a pull request.
